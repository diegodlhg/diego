# Simple Calculator

This repository contains a simple command-line calculator written in Python.

## Usage

Run the script using Python and specify the operation followed by two numbers:

```bash
python calculator.py <operation> <a> <b>
```

Where `<operation>` can be one of:

- `add`
- `subtract`
- `multiply`
- `divide`

Examples:

```bash
python calculator.py add 4 2      # Outputs: 6
python calculator.py divide 10 2  # Outputs: 5
```
