# Simple Calculator

This repository contains a basic command line calculator written in Python.

## Usage

Run the script with two numbers and an operation:

```bash
python3 simplecalc.py 3 + 4
```

If you omit the arguments, it will prompt you interactively:

```bash
python3 simplecalc.py
```

Supported operations: `+`, `-`, `*`, and `/`.
